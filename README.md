# Timeless - "Relive the moments."

## Welcome! If you want to explore our project in detail, please:


### 1. Clone the project

```
git clone https://github.com/ch3ngZ/Timeless.git
```

### 2. Go to the project directory

```
cd Timeless
```

### 3. Install necessary dependencies

```
npm install
```

### 4. Start the server

```
npm run dev
```
